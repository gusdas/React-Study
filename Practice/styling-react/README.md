## 1장 리액트 시작

# 1. 리액트 특징

    - 이전 DOM과 Virtual DOM을 비교하여 변경 된 부분만 실제 DOM에 업데이트 하는 방식이다.
    - 프레임 워크가 아닌 라이브러리로 뷰만 담당하기 때문에 개발자에 맞게 라우팅, Ajax, 상태관리, 데이터 모델링 등등 원하는 라이브러리를 사용해야 된다.
